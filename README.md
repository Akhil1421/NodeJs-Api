# NodeJs-Api

## Routes

### Get Stats of Blogs (GET)

``` baseUrl/api/blog-stats```

Command:
```
curl --location 'baseUrl/api/blog-stats' 
```
### Get Blogs Based On Search Query (GET)

``` baseUrl/api/blog-search```

Command:
```
curl --location 'baseUrl/api/blog-search?query=keyword_name' 
```
